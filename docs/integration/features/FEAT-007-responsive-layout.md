# FEAT-007 Responsive Layout

## Scope
Ensure lane visualization and controls remain usable on desktop and tablet.

## Dependencies
- Chunks: CH-004, CH-006, CH-008
- Integrations: INT-006, INT-007, INT-008

## Build steps
1. Add responsive breakpoints in SCSS.
2. Keep controls and lanes readable at target widths.
3. Validate overflow and label truncation behavior.

## Acceptance criteria
- [ ] Desktop layout is stable and clear.
- [ ] Tablet layout remains fully functional.
- [ ] No overlap between moving objects and controls.
